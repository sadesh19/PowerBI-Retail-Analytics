
## Project Overview
This interactive dashboard provides a comprehensive analysis of retail performance using the Superstore Sales dataset. It focuses on key financial metrics and efficiency indicators to drive business decisions.

## Features & KPIs
* **Financial Performance:** Tracked via Total Revenue and Total Profit.
* **Operational Efficiency:** Visualized through Average Order Value and Profit Margin.
* **Product Insights:** Identifies the Top 10 Products by revenue.
* **Geographical Trends:** A mapped distribution of sales across the United States.
* **Interactive Filtering:** Dynamic slicers for Category and Order Date ranges.

## Tools Used
* Power BI Desktop
* DAX (Data Analysis Expressions) for custom KPIs
* Power Query for data cleaning
